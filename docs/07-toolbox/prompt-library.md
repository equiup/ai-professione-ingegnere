# Prompt library

Raccolta di prompt professionali per ingegneri.

## Analisi di una norma tecnica

```text
Agisci come assistente tecnico per un ingegnere. Analizza il seguente testo normativo e restituisci:
1. ambito di applicazione;
2. obblighi principali;
3. requisiti tecnici;
4. rischi di non conformita;
5. checklist operativa per la verifica.
Non inventare riferimenti non presenti nel testo. Evidenzia eventuali dubbi interpretativi.

Testo: [incolla testo]
```

## Relazione tecnica

```text
Aiutami a predisporre una bozza di relazione tecnica professionale sul seguente tema: [tema].
La relazione deve contenere: oggetto, quadro di riferimento, metodologia, analisi tecnica, conclusioni, limiti e allegati.
Mantieni un tono tecnico, chiaro e verificabile. Evidenzia i punti che richiedono verifica da parte del professionista.
```
